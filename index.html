A bit more about me

Before starting at USD, I served four years in the Marine Corps, reaching the rank of noncommissioned officer. That experience put me inside DoD information environments governed by RMF frameworks long before I knew what a SIEM was. Coming out, I enrolled at USD to formalize what I'd been doing operationally and push deeper into the technical side.

Today I spend time in Splunk writing SPL queries, running nmap scans, and scripting Python tools that automate security workflows — including a project that connects Splunk's REST API to an LLM for automated SSH brute-force alert summaries. I'm working toward my Security+ and building toward a career in threat detection and incident response.
